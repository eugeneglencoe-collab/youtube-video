⚠ Note sur l'assemblage vidéo (Remotion)
L'étape d'assemblage nécessite Remotion, qui tourne en Node.js côté serveur.
GitHub Pages est 100% statique — il ne peut pas exécuter Node.js.

Solution recommandée : Déploie un micro-backend Remotion gratuitement sur Render.com ou Railway (plan gratuit).
Ce site appelle alors ton backend via une simple requête HTTP. Je peux te générer le code du backend dès que tu es prêt.
